# 🚀 Quick Reference - Make Money with Polymarket Miner

## What It Does
- Fetches fresh markets from Polymarket
- Analyzes sentiment (BULLISH/BEARISH/NEUTRAL) using AI
- Finds trading opportunities where price ≠ sentiment

## How to Make Money

### 1. Get Opportunities (Daily)
```bash
cd polymarket-miner
source venv/bin/activate
export OPENROUTER_API_KEY='your-key-here'
python3 src/find_opportunities.py
```

### 2. Trade on Polymarket
- Go to [polymarket.com](https://polymarket.com)
- Search for the market
- Place bet based on recommendation

### 3. Track Results
- Start small ($5-10 per trade)
- Record wins/losses
- Learn from experience

## Core Scripts
- `python3 src/main.py` - Fetch markets
- `python3 src/analysis.py` - Analyze sentiment
- `python3 src/find_opportunities.py` - Find trades ⭐

## Money-Making Strategy
```
Market price ≠ Sentiment = Opportunity
→ Bet on mispriced markets
→ Profit when price corrects
```
